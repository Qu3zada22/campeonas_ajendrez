# Reinas del Tablero

Línea de tiempo interactiva de las 17 campeonas mundiales de ajedrez femenino
(1927–presente), construida con React + Vite.

Cada tarjeta incluye retrato, país, años de reinado, rival por el título,
detalles del match y un dato curioso sobre su estilo de juego. Al final de la
página se explican las pequeñas discrepancias entre las fuentes consultadas.

## Fuentes

- [Chess.com — Women's World Chess Champions](https://www.chess.com/article/view/womens-world-chess-champions)
- [Federación de Ajedrez de Aragón — Campeonas del mundo (PDF)](https://www.ajedrezaragon.org/uploads/33029a9e-a372-4801-926b-324a0b499a7c/Campeonas_del_mundo.pdf)
- [FIDE — Women's World Chess Championship 2025](https://womenworldchampionship2025.fide.com/)
- Wikipedia (biografías individuales de cada campeona) y FIDE Chess Museum

## Desarrollo

```bash
npm install
npm run dev
```
